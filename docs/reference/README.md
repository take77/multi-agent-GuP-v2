# docs/reference/

本ディレクトリは `projects/` 配下のプロジェクト成果物のうち、GuP-v2 本体リポジトリで共有すべきものの**スナップショット**を置く場所です。

## 位置付け

- **原本（真）**: `projects/<project_id>/docs/` 配下（git-ignored）
- **スナップショット（本ディレクトリ）**: git 追跡対象、複数端末間で `git pull` 一発で共有可能

`projects/` 配下は `.gitignore` により除外されているため、プロジェクト横断で参照したい成果物は本ディレクトリに手動コピーして共有します。

## 更新ルール

1. **原本を変更したらスナップショットも同期**: 原本（`projects/` 配下）の内容を更新した場合、本ディレクトリのコピーも更新して同一 PR で commit すること。
2. **スナップショット側で直接編集しない**: 本ディレクトリはあくまで参照用。編集は必ず原本に対して行う。
3. **追加時は本 README の収録表に追記**: 新しいファイルを追加する場合、このファイルの「収録ファイル」表に出自・原本パス・最終更新日・概要を記載すること。
4. **削除時は表からも削除**: ファイル削除時は収録表の該当行も削除する。
